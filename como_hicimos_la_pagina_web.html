¿Cómo hicimos la pagina web?

Decidimos crear una pagina web porque nuestro grupo ya tenía algo de experiencia con la programación.
Utilizamos herramientas como HTML, CSS, JavaScript y GitHub Pages. HTML significa "Hyper Text Markup Language".
Esta herramienta se usa para decirle al navegador qué mostrar al usuario cuando ingresó a la página web.
CSS significa "Cascading Style Sheets". Se utiliza para diseñar la página web y darle una vista única.
JavaScript se utiliza para dar vida a los elementos interactivos de la página. En nuestra web se utilizaba
para realizar la navegación. Las GitHub Pages se utilizaron para publicar nuestro sitio en la web.
